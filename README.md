# Visualizacion-de-Datos--Basketball

# Fuentes de datos

## nba_advanced_2025.csv

**Origen:** Basketball-Reference.com
**URL:** https://www.basketball-reference.com/leagues/NBA_2025_advanced.html
**Descripción:** Estadísticas avanzadas de todos los jugadores NBA en la temporada 2024-25.
Incluye métricas como PER (Player Efficiency Rating), TS% (True Shooting),
USG% (Usage Rate), BPM (Box Plus/Minus), entre otras.
**Descarga:** Tabla exportada directamente desde el sitio en formato CSV.
